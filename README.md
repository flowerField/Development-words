List some common words used in program development

> [developer]()      开发人员

> implementation 实现

> [iterator](http://dict.youdao.com/search?q=max)        迭代器、迭代程序、遍历器(接口)

> promise        承诺、允诺、给人以...希望

> compile        编译

> template       模板

> replace        替换

> script         脚本

> parse          解析

> tag            标记、标签等

> type           类型

> test           测试

> length         长度

> element        元素

> node           节点

> min            最小值
